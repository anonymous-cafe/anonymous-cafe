## 🐳 Docker Setup (Recommended)

```dockerfile id="docker-pro"

WORKDIR /app
COPY package*.json ./
RUN npm install

COPY . .

EXPOSE 3000
CMD ["npm", "start"]
```

Run:

```bash id="docker-run"
docker build -t anonymous-cafe .
docker run -p 3000:3000 anonymous-cafe
```
